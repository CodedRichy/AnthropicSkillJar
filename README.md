# AnthropicSkillJar

A personal learning repository documenting my journey through the [Anthropic Academy](https://anthropic.com/academy).

## About

This repo serves as a running record of the skills, concepts, and projects I explore while working through Anthropic's educational materials. Think of it as a skill jar — each lesson or exercise adds something new.

## What's Inside

As I progress through the academy, this repository will grow to include:

- **Notes & summaries** — key takeaways from each module
- **Code exercises** — hands-on practice with the Claude API and related tools
- **Projects** — small end-to-end projects that apply what I've learned
- **Experiments** — prompting strategies, model comparisons, and other explorations

## Goals

1. Build a solid understanding of large language models and how to work with them responsibly.
2. Get hands-on experience with the Claude API.
3. Develop best practices for prompt engineering and AI-assisted development.
